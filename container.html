Демонстрация Bootstrap Grid System

<div class="container">
  <div class="row">
    <div class="col">...</div>
    <div class="border col">...</div>
    <div class="bg-success col">...</div>
    <div class="border col-3">...</div>
    <div class="bg-success col-9">...</div>
  </div>
</div>

Результат:

class="col"
Автоматическая ширина (равномерное распределение)
class="border col"
Колонка с рамкой
class="bg-success col"
Колонка с зеленым фоном
class="border col-3"
Ширина: 3 колонки из 12
class="bg-success col-9"
Ширина: 9 колонок из 12

Объяснение классов:

  • .container - центрирующий контейнер с отступами
  • .row - строка для размещения колонок
  • .col - автоматическая колонка (равномерное распределение)
  • .col-3 / .col-9 - колонки с фиксированной шириной (3/12 и 9/12)
  • .border - добавляет рамку вокруг элемента
  • .bg-success - устанавливает зеленый фон
